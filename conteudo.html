Conteúdo



HTML



HTML (Hypertext Markup Language) é a linguagem usada para criar a
estrutura básica de páginas web. Ele utiliza elementos chamados "tags" para organizar
o conteúdo da página, como 'p' para parágrafos e 'h1' a 'h6' para títulos. As tags
podem ter atributos, como o atributo 'href' na tag 'a' para criar links. Um documento HTML
começa com '!DOCTYPE html', seguido pelo 'html' como elemento raiz. O 'head'
contém informações sobre o documento, como o 'title' para o título da página,
e o 'body' contém o conteúdo visível da página, como texto, imagens e vídeos.
Combinando tags e atributos, é possível criar páginas web com a estrutura e o
conteúdo desejados.



CSS



CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e dar estilo às páginas web
criadas com HTML. Enquanto o HTML define a estrutura e o conteúdo da página, o CSS controla
a aparência e o layout desse conteúdo.



JAVASCRIPT



JavaScript é uma linguagem de programação amplamente usada para criar interatividade em páginas web.
Enquanto o HTML é usado para estrutura e conteúdo, e o CSS para estilo, o JavaScript é
responsável pela lógica e comportamento dinâmico das páginas web.



BOOTSTRAP



Bootstrap é um framework front-end popular usado para criar interfaces web responsivas
e com aparência profissional de forma rápida e fácil. Ele é construído com HTML, CSS e JavaScript,
e fornece uma coleção de componentes e estilos predefinidos que podem ser facilmente
utilizados em um projeto web. Com o Bootstrap, você pode criar rapidamente layouts e designs
consistentes, independentemente do dispositivo em que a página é visualizada.
